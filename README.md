## 项目介绍  
**島語** 是为 [X岛](https://www.nmbxd1.com/Forum) 网站开发的一款移动端应用。  
X岛是一个拥有多种论坛（forum）的社区，用户可以匿名发表文章（称为“串”）。  
**島語** 提供了更加便捷的论坛浏览、发串和管理体验。  

### 应用功能  
- 登录（前台匿名，后台实名，通过饼干实现）  
- 查看不同版面的串内容  
- 支持图片预览和版面切换
